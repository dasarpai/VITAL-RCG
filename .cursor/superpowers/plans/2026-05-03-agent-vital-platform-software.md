# VITAL Platform Software Agent — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Deliver a **reference implementation** where an organisation can **plug in connectors**, aggregate to VITAL’s stable IDs, run **rule evaluation stubs**, and **generate dashboards** from declarative config—suitable as the kernel future agents wrap.

**Architecture:** **vital-core** (schemas + domain logic, no IO), **vital-api** (HTTP ingest + query), **vital-dashboard** (static site generator or SPA reading JSON bundles). SQLite for demo tenancy; swap for Postgres later without changing core.

**Tech Stack:** Python 3.11+, FastAPI, Pydantic v2, SQLite (SQLModel or raw SQL), pytest; dashboard: Vite + React + TypeScript **or** plain static HTML generated by Python (choose React if team prefers one SPA). This plan assumes **FastAPI + React (Vite)** for realism.

**Agent charter:** Owns **schemas, APIs, UI views, CI**—not enterprise legal text or research hypotheses.

```python
class FunctionRecord(BaseModel):

class FunctionRegistry(BaseModel):
```

```python
from fastapi import FastAPI
from pydantic import BaseModel

from vital_core.models import MonthlyAggregateSnapshot

app = FastAPI(title="VITAL API")
_STORE: dict[tuple[str, str], MonthlyAggregateSnapshot] = {}


class IngestPayload(BaseModel):
    snapshots: list[MonthlyAggregateSnapshot]


@app.post("/v1/tenants/{tenant_id}/snapshots")
def ingest(tenant_id: str, payload: IngestPayload):
    for snap in payload.snapshots:
        if snap.tenant_id != tenant_id:
            raise ValueError("tenant mismatch")
        _STORE[(tenant_id, f"{snap.period}:{snap.function_id}")] = snap
    return {"accepted": len(payload.snapshots)}


@app.get("/v1/tenants/{tenant_id}/dashboard-bundle.json")
def bundle(tenant_id: str):
    items = [v for k, v in _STORE.items() if k[0] == tenant_id]
    items.sort(key=lambda s: (s.period, s.function_id))
    return {"tenant_id": tenant_id, "snapshots": [i.model_dump() for i in items]}
```

```tsx
import { useEffect, useState } from "react";

type Snapshot = {
  tenant_id: string;
  period: string;
  function_id: string;
  tier: string;
  participation_rate: number;
  signal_stress_index: number;
};

export default function App() {
  const api = import.meta.env.VITE_VITAL_API ?? "http://127.0.0.1:8000";
  const tenant = import.meta.env.VITE_VITAL_TENANT ?? "t1";
  const [rows, setRows] = useState<Snapshot[]>([]);

  useEffect(() => {
    fetch(`${api}/v1/tenants/${tenant}/dashboard-bundle.json`)
      .then((r) => r.json())
      .then((j) => setRows(j.snapshots ?? []))
      .catch(() => setRows([]));
  }, [api, tenant]);

  return (
    <div style={{ fontFamily: "system-ui", padding: 24 }}>
      <h1>VITAL Dashboard (demo)</h1>
      <table cellPadding={8}>
        <thead>
          <tr>
            <th>Period</th>
            <th>Function</th>
            <th>Tier</th>
            <th>Participation</th>
            <th>Stress</th>
          </tr>
        </thead>
        <tbody>
          {rows.map((s) => (
            <tr key={`${s.period}-${s.function_id}`}>
              <td>{s.period}</td>
              <td>{s.function_id}</td>
              <td>{s.tier}</td>
              <td>{(s.participation_rate * 100).toFixed(0)}%</td>
              <td>{s.signal_stress_index.toFixed(2)}</td>
            </tr>
          ))}
        </tbody>
      </table>
    </div>
  );
}
```
